### Surprises / learnings

- **arXiv response is XML (Atom);** the skill body says "Parse responses (XML for arXiv via xmllint or by hand)". A small Python `ET` script proved the simplest route. Worth recording as a recommended path inside the SKILL.md for future skill-runs (currently the body just lists xmllint as a hint).
- **Author-institution scoring is awkward when the roster is mixed.** ActCam has Torr (Oxford) and Laptev (MBZUAI/Inria) but most of the team is from Kinetix (an industry studio). Scoring this as "famous-author paper" would over-credit the senior names. Settled on 11/15 — meaningful but not maxed. The skill body should note that mixed-roster papers should be scored on the median, not the maximum.
- **"We outperform in all cases except 3C" caption discrepancy.** ActCam's Table 1 shows Uni3C wins on Background Consistency and Temporal Consistency, contradicting the caption's claim. Caught this only after reading the table; abstract-only screening would have missed it. Reinforces the value of Stage 2 for the top tier — and is exactly the kind of "penalize hype, mention insufficient evidence" call PRD §13 asks for.
